This is a Blog Application written in python.'webapp' framework of GAE is used in this application.
All the users with adminitrator access can write and post and can create new blogcategories.

Currently It's been integrated with Google BUZZ where users can see as well as create new BUZZ posts.

To get this source working for BUZZ you'll be reuired a 'consumer.py' file with CONSUMER_KEY and CONSUMER_KEY_SECRET values
set to whatever key and secret you got when you registered your application with google.
